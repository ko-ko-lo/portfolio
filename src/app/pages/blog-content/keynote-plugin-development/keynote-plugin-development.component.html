The Emotional Rollercoaster

If you’ve ever worked on a personal tech project, you know the ride.

At first I was optimistic: “This will be fun. How hard can it be?” Then came the frustration: debugging, weird API behavior, error messages that made me question my life choices. But eventually, after trial and error, something clicked. And when the plugin finally worked, I felt victorious: “I built this thing!”

It reminded me that learning new tech is rarely linear. It’s try → fail → learn → repeat.

Why Plugins?

For those who don’t know: plugins in Figma are little helpers. They export colors, insert icons, generate data: small tools that make design work faster or easier. Basically, if Figma doesn’t do it by default, there’s probably a plugin for that.

I’ve created several plugins, but for this talk I chose to share three of them:”

  • Hexify - turns color collections into code (SCSS, CSS, JSON)
  • JSONify - exports content from Figma into structured JSON data
  • Pattern Hearts - a tiny plugin to spread a little joy through design

Each one came with its own set of challenges and lessons.

Hexify: Learning by Doing

Hexify was my first plugin. It sounded simple: export Figma colors into code. But starting out, I had almost no knowledge of the Figma API. I skimmed the docs, watched a few YouTube videos, and then thought: “Alright, let’s try to build something real.”

Step by step, I connected the dots. I discovered how to access variables, filter colors, and convert RGB values into Hex. Just when I thought I was done, I realized semantic tokens were referencing other colors: another dot to connect.

JSONify: Automating My Own Workflow

My second plugin grew out of a personal project: Open Stillness, a platform I’m building for Meditation, Breathwork, and Yin-Style yoga.

Each piece of content, a pose, a meditation, a technique, is documented as a card in Figma. At some point, I needed a way to turn all those cards into structured data.

Enter JSONify

It meant diving into Figma’s node tree with a Depth-First Search algorithm, finding text layers buried inside frames and groups. I discovered I could use layer names as keys, exclude layers with a naming rule, and even split line breaks into arrays.

That little mapping trick, from design layer to JSON output, unlocked everything. Suddenly, I wasn’t just designing content, I was building data structures straight out of Figma.

Pattern Hearts: Joy Through Design

After Hexify and JSONify, I wanted something lighter. Inspired by an old art project where I drew patterned hearts on paper, I built Pattern Hearts. It is a plugin to add patterned hearts (and rainbow gradients!) to designs.

It wasn’t just playful, it was also personal. Each heart stood for openness, empathy, and individuality. Pattern Hearts is a joyful plugin made to share a little love.

Takeaways: From Chaos to Creation

Looking back, these three plugins taught me more than just the Figma API:

  • Start small. Even tiny projects can spark real growth.
  • Embrace chaos. Trial and error is not failure, it’s the process.
  • Follow your curiosity. The best tools often emerge from personal needs.
  • Bring joy. Tech isn’t only about efficiency, it can also spread a little love.
